# visualization_final_project
只需要看final_work
# 使用教程
安装node.js

打开cmd

cd 到 final_work/visual-data-master

输入npm run serve

打开另一个cmd

cd 到 final_work/visual-server-master

输入node index.js

打开浏览器，输入网址localhost:8080

如果画面出现缩放问题。请调整你的画面大小（Ctrl+鼠标滚轮）
